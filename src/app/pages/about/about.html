Sobre o Projeto

Angulibrary é uma biblioteca virtual criada para tornar a leitura mais acessível por meio de uma plataforma simples, moderna e intuitiva.

Este projeto foi desenvolvido como parte de uma jornada de aprendizado em desenvolvimento web, com foco em arquitetura limpa, performance e experiência do usuário.
