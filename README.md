## Angulibrary 📚

Angulibrary is a modern, interactive book search application built with Angular 21 using standalone components. Users can search for books, filter by author, year, or genre, view a “Book of the Day,” and manage a favorites list. The app includes light/dark theme switching, pagination, and a responsive design.

---

## 🚀 Tech Stack

Angular 21
TypeScript 
Bootstrap 
CSS3

## 🛠️ Getting Started
1️⃣ Clone the repository

```bash
git clone https://github.com/yourusername/angulibrary.git
```

2️⃣ Navigate to the project folder
```bash
cd angulibrary
```

3️⃣ Install dependencies
 ```bash
npm install
```

4️⃣ Run development server
```bash
ng serve
```

The app will be available at:

http://localhost:4200
